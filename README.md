## About Node.js
Node.js is a **JavaScript runtime built on Chrome's V8 engine**. It allows developers to run JavaScript on the server side, making it possible to build fast and scalable backend applications.

**Key Features:**
- Event-driven, non-blocking I/O
- Single-threaded but highly scalable
- Huge ecosystem with NPM (Node Package Manager)
